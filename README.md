Phishing Email Forensics Console
A web-based phishing email analysis tool developed using Python and Flask.

Features
- Email header analysis
- URL inspection and extraction
- Detection of phishing indicators
- User-friendly web interface

Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py

Project Purpose: This project helps analyze suspicious emails by examining headers, links, and other indicators that may suggest phishing activity.
